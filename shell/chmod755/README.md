# chmod755

현재 디렉토리의 모든 항목에 `755` 권한을 부여하는 스크립트.

## 하는 일

```sh
chmod 755 ./*
```

스크립트 파일들에 실행 권한이 빠졌을 때 한 번에 부여하는 용도.

## 실행

```sh
# 로컬 (실행 위치의 ./* 에 적용)
bash shell/chmod755/chmod755.sh
```

## 주의

- **현재 디렉토리 기준**으로 동작한다 — 실행 위치를 반드시 확인할 것 (원격 `curl` 실행 시 의도치 않은 위치에 적용될 수 있음)
- `755`는 소유자 rwx, 그룹·기타 r-x. 민감 파일엔 과한 권한일 수 있다
